# COSC 481: Playful Thinking, Serious Coding

Spring 2026

Instructor: Professor Elodie Fourquet 

## Course Objectives

1. Create games from ground-up without game engines
2. Use coordinate systems, linear algebra, physics simulatoin to implement a primitive game engine.
3. Understand game loop, event handling and graphic rendering to design engaging games.

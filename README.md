# COSC 481: Playful Thinking, Serious Coding

**Spring 2026**

**Instructor:** Professor Elodie Fourquet

## Course Objectives

1. Create games from the ground up without game engines
2. Use coordinate systems, linear algebra, and physics simulation to implement a primitive game engine
3. Understand game loops, event handling, and graphics rendering to design engaging games
